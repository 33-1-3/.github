## 🤔 기획 의도
- LP 를 수집하는 사람들이 소장 LP를 관리할 수 있는 편리한 아카이빙 서비스가 따로 없다.
- 좋은 가독성과 UI로 사람들의 LP 수집욕을 자극함과 동시에 수집한 LP 목록을 다른 사람에게 공유하는 아카이빙 서비스를 만들고자 한다.
